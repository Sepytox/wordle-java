# Wordle auf Konsole 🟩

Wordle mit 6 Versuchen

| Symbol | Bedeutung |
|--------|-----------|
| `[✓]`  | buchstabe an richtiger Position |
| `[~]`  | Buchstabe im Wort, falsche Position |
| `[ ]`  | Buchstabe nicht im wort |

## Voraussetzungen

- Java JDK 21

## Starten

```powershell
javac Main.java
java Main
```

> **Windows PowerShell:** Falls die Zeichen nicht richtig angezeigt werden,
> einmalig vorher eingeben:
> ```powershell
> [Console]::OutputEncoding = [System.Text.Encoding]::UTF8
> ```
